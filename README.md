Welcome to Solstice Avelo Airlines Mobile Redesign

Solstice  consists of six members who have redesigned a mobile experience for Avelo Airlines for an interaction design project.  
The main goal of this project  is to translate a high-fidelity Figma prototype into a functional, responsive app that matches the original visual design and interaction.
We focused on accessibility, usability and general interaction design.

Tech Stack
- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS
